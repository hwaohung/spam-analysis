## Assignment 1: spam analysis

- 下載[SPAM ARCHIVE](http://untroubled.org/spam/)從2010~2015年的資料
- 使用LogStash進行資料拋轉的工作
- 匯入至ElasticSearch
- 使用Kibana做統計圖表
- 取出5種屬性，轉成data matrix
- 部署分散式Hadoop，執行K-means演算法
